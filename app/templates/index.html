Meld components consist of a simple class and a template.

Meld components define the data attributes and functions that the template will use.

            
meld/components/search.py

            
                from flask_meld import Component
                from app.models import User

                class Search(Component):
                    search = ""

                    @property
                    def users(self):
                        return User.query.filter(User.name.contains(self.search)).all()
