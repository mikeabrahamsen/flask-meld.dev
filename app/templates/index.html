Flask-Meld: A full-stack framework that enables dynamic frontends using Python and the Jinja2 templating engine.

Meld is a passion project. I want to show how it works and how it can help you create dynamic web interfaces in pure Python.

  • - Less context switching.
  • - No need to write JavaScript.
  • - More fun!

Meld utilizes WebSockets and morphdom to create server-rendered HTML and swap out DOM elements without refreshing the page.
